# dijital-sinav-sinifi-degerlendirme
Bu çalışma, Dijital Sınav Sınıfı uygulamasının etkililiğini değerlendirmek amacıyla geliştirilen bir değerlendirme anketini içermektedir. Anket, öğrenciler ve öğretmenler başta olmak üzere paydaş görüşlerini toplamak üzere hazırlanmış; Jones (2005) Teknoloji Planlama Modeli doğrultusunda uygulama sonrası değerlendirme aşamasına odaklanmıştır.
